# colaberry-build-system

Minimal scaffold for the Colaberry Internship Build System.

Goals
- Provide a clean local development layout for Python projects
- Encourage disciplined build steps and reproducible setup

Quick start (Windows PowerShell)

```powershell
python -m venv .venv
.venv\\Scripts\\Activate.ps1
pip install -r requirements.txt
pytest
```


Layout
- src/ - Python package source
- tests/ - pytest tests
- requirements.txt - minimal developer dependencies
